What this tool does

      
Drop in a Design Agreement: Scope Outline .docx and this extracts the project header, sections, items (with level 0/1 nesting preserved), allowances, and TBDs into the structured envelope that Phase 1 CA generation expects.

      
Structure is parsed deterministically from the .docx. If you've saved an Anthropic API key in Settings, each scope item also gets a classification and probable CA trade divisions assigned by Claude — both editable in the review UI below.
