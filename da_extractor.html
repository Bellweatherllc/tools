What this tool does

      
Drop in a Design Agreement .docx and this extracts the project header, design scope, fee and payment schedule, and notable terms into the structured envelope that the CA Writing Tool expects.

      
Header and fee fields are parsed deterministically from the .docx. If an Anthropic API key is set in Settings, Claude then extracts design scope, deliverables, and terms — all editable in the review UI below.
