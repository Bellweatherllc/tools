Automated (normal flow)
New Trello cards with the EST REVIEW label trigger a Make automation that extracts the card description via Claude, maps fields to the CORE schema, and writes the record directly to SharePoint — nothing needed here. Projects appear in the list automatically within a few minutes.
Manual Trello import
Use the form below to pull cards directly from Trello via the API — useful for catch-up imports, backfilling older cards, or cards that didn't go through the automation. Connect with your Trello API key and token, then select the cards to bring in.
Manual entry
For projects with no Trello card, enable Admin mode (toggle in the header) then click + New Project in the toolbar. Fill in the required fields and save — the record lands in SharePoint immediately. Use the detail panel to fill in remaining fields.
