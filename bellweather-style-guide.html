01 — Typography

Type Hierarchy

Bellweather's typography pairs TT Norms Pro (a geometric sans-serif by TypeType) with Swear Text and Swear Display (high-contrast serifs by Type Foundry). The interplay between the two families creates a warm, editorial character that balances professionalism with personality.

05 — Reference

CSS Variables

Copy-paste ready. Every CORE tool should include these variables for brand consistency. 

:root {
  /* Primaries */
  --limestone: #282828;
  --cotton: #F6F2EE;

  /* Secondaries */
  --navy: #263860;
  --slate: #9091A0;
  --mahogany: #54423E;
  --teak: #B7A9A3;
  --concrete: #D2D5D8;
  --glass: #EDF5F9;
  --linen: #E5E1D8;
  --marble: #FFFFFF;

  /* Font Stacks */
  --font-sans: 'TT Norms Pro', -apple-system, BlinkMacSystemFont, sans-serif;
  --font-serif-text: 'Swear Text', Georgia, serif;
  --font-serif-display: 'Swear Display', Georgia, serif;
}
